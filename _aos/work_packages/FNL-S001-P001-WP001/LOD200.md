# LOD200 — Family Newsletter Edition #1 Product Contract

**The small gate everything hangs on.** Defines what edition #1 MUST contain, the "good-enough" acceptance bar, and the weekly cost ceiling. Content → LOD200 + cheap validation (per standing law). All code/templates/config/installs that build this are separate LOD400 WPs (P002–P004) that derive FROM this contract.

## 1. Intent (LOD100)
Revive the weekly family newsletter for בית ולד as a real, recurring edition. Edition #1 = the magic moment: a genuine, personalized edition reaches the **test group (Nimrod + Tzlil)**. Scope decision (team_00, 22.7): **FULL section set**, completeness over speed — ~10-day timeline accepted.

## 2. MUST-contain — edition #1 section set (FULL)
Ordered, per STYLE_GUIDE §4 + the 3 revival additions. Bilingual RTL, comic template (unchanged).
1. **שער / Cover** — masthead "Family Newsletter" / "בית ולד", edition #, mascot bubble
2. **פתיח / Opener** — warm, names each member; **tone-aware** (see §5 — grandfather Rami's recovery)
3. **פס משפחה / Family Strip** — chip per member + item count
4. **מזג אוויר / Weather** — **Pardes Hanna ONLY** (Basel REMOVED — Shaked home)
5. **פינה אישית ×5** — one per member (Nimrod, Michal, Shaked[EN], יויו, צליל); **every member ≥1 item** (hard rule)
6. **גשר גילוי / Discovery** — cross-member recommendation
7. **🍿 מה רואים השבוע / Viewing** — 1 family pick + 1 rotating personal pick; **availability + Hebrew-subtitles verified** (Netflix/Prime IL); calibrated from real harvest (Ghibli in Tzlil's list, Avatar co-watch)
8. **🍽️ שולחן שישי / Family item** — one conversation-starter for everyone + an open question (Minecraft server, Death-Gate reading, orchard season…)
9. **חידה / Puzzle** — for Tzlil, **hard** (gifted-math level); last week's answer
10. **👨‍👩‍👧 מהמשפחה המורחבת / Extended family** — 1–2 items (e.g. Hadar's chamber-music win); **public-only, pointer+link not copy, loving tone**
11. **היום בהיסטוריה / Today in history**
12. **סקר / Survey** — one question; teaser caption invites group reply (WAHA poll mechanism = Phase C)
13. **סגירה / Closer** + **Footer** — family emoji strip, edition #, **"עורכת: צליל"** editor credit, "by AgentsOS @ nimrod.bio"

## 3. Happy-path acceptance criterion ("good-enough" bar — buildable checklist)
Edition #1 PASSES when ALL hold:
- [ ] Build completes without fatal error; rendered HTML ≥ 30KB
- [ ] All 13 sections present; all 5 members have ≥1 item; puzzle present
- [ ] Weather = Pardes Hanna only (no Basel)
- [ ] Viewing picks verified available + Hebrew subs
- [ ] Teaser image generated; uploads to uPress → **HTTP 200** at the public URL
- [ ] WhatsApp teaser (image + caption + link) delivered to the **test group** (Nimrod + Tzlil)
- [ ] Runtime cost < cost_cap (§6)
- [ ] Content passes the §5 cheap-validation checklist
**Content quality ~70% on-target is ACCEPTED** — the loop (Phase B) refines. "Precise + fast from static profiles" is explicitly NOT the bar.
