## Costuming principle (Tranche B)
The **same Skipper Cat wears a per-topic costume** — e.g. an **architect cat**
(hard-hat + T-square) for Michal's architecture; a **circus cat** (in an aerial hoop)
for Yoyo. Keep the cat's core identity CONSTANT (hat, face, white fur, palette, line
weight); only the costume / prop / scene changes.

## Topic → scene → character (per-edition art selection, reference)
Sailing → sea + wheel + sails (Nimrod). Circus → tent + trapeze (Yoyo). Architecture →
drafting + T-square (Michal). Science/reading → books + test-tubes (Shaked). Baking/math
→ oven + numbers (Tzlil). (Full 10-row table in `SVG_MODULE_SPEC.md` §4.)
