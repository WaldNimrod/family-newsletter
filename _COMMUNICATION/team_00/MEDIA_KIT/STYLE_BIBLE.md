# STYLE BIBLE — Family Newsletter visual system (LOCKED)

Apply to EVERY generated asset. Consistency beats novelty.

## Art direction (one line)
**"Quentin Blake meets Hergé"** — loose, energetic, slightly scratchy hand-INK linework
(Blake) + clean flat "ligne claire" color fills (Hergé).

## Rendering rules (hard)
- BOLD, slightly uneven black ink **outlines** (`#2c2c2c`).
- **FLAT solid colors ONLY.** No smooth gradients, no soft shading, no airbrush, no 3D,
  no photorealism, no gloss/bevel.
- Shadows: a single **flat offset shadow with ZERO blur** (comic style).
- Mood: warm, playful, family-friendly, expressive.

## Palette (use ONLY these hex)
- **Core:** ink `#2c2c2c` · warm-cream world `#fdf6e3` · panel white `#ffffff` ·
  Skipper red `#c0392b` · wood `#8B4513` · sun yellow `#f39c12` · kite red `#e74c3c`.
- **Members (each has a color + accent):** נימרוד blue `#2471a3`/`#1a5276` · מיכל green
  `#27ae60`/`#1e8449` · שקד purple `#7d3c98`/`#6c3483` · מעיין·יויו red `#c0392b`/`#a93226`
  · צליל orange `#e67e22`/`#ca6f1e`.
- **Member tint backgrounds:** nimrod `#e6f0fa` · michal `#e6f5ed` · shaked `#ece6f5` ·
  maayan `#fce8ec` · tzlil `#fef3e2`.

## Comic mechanics (for scenes / backgrounds)
- Halftone-dot texture allowed on backgrounds (small dots ~`#e8dcc8`).
- Flat ink borders, thick→thin. Flat offset drop-shadows, zero blur. Rounded-corner feel.

## Backgrounds
- **Character assets → FULLY TRANSPARENT** (PNG alpha). No scenery, no ground line, no frame.
- **Scene / hero assets → opaque**, flat-color scene; leave open sky for text (add NO text).

## THE MASCOT — "Skipper Cat" (draw IDENTICALLY, every time)
- Friendly cartoon domestic **CAT**, upright/bipedal, full body.
- Fur **WHITE** (`#ffffff`); a few warm-red (`#c0392b`) stripe accents on the tail.
- Wears a **TALL stovepipe hat, horizontal RED (`#c0392b`) + WHITE stripes** (Cat-in-the-Hat hat) — this is the constant identity.
- Face: large round friendly eyes, small pink triangular nose, whiskers, warm closed-mouth smile.
- Proportions: cute, child-friendly — big head, small rounded body.
- Signature prop: a **brown wooden ship's wheel** (`#8B4513`) — present in the hero, optional in small poses. Nautical/skipper spirit.

## Consistency lock
Same cat · same hat · same palette · same line-weight in EVERY asset. When unsure,
match the reference render — not imagination.
